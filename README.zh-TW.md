# 全球人口統計模擬引擎

**語言：** [English](README.md) | 繁體中文

一個互動式 React 單頁應用程式，模擬 **9 個國家與地區**從 **2025 年至 2100 年**的人口變遷軌跡。調整總和生育率（TFR）與淨移民人數，探索不同政策選擇如何影響撫養比與經濟未來。

**[→ 部署網頁](https://lawrencehwang.github.io/taiwan-demographics/)**

> **AI 協作說明：** 本專案由 Gemini（Google）與 [Claude](https://claude.ai)（Anthropic）共同協作完成。

## 關於本專案

本模擬工具採用標準的**世代組成人口模型（Cohort-Component Method）**，預測不同政策槓桿對各國未來人口的影響。

核心指標為**撫養比**——每 100 名勞動年齡成人所對應的受撫養人口數（兒童加上老年人）——並對應從「健康人口紅利」到「系統崩潰」的各個閾值區間。

每個國家均包含：

- 以各國統計局與聯合國 WPP 2024 為基礎的 2025 年年齡分佈
- 各國實際 TFR 與淨移民預設值
- 適配的死亡率曲線（已開發國家預期壽命 ~80 歲以上 vs. 高生育率國家 ~62 歲）

## 功能特色

- **9 個國家／地區** — 一鍵切換，模擬自動重置為各國 2025 年基準值
- **即時世代模擬** — 逐年迭代 101 個年齡組，套用死亡率、生育率與移民率
- **固定 vs. 動態生育率情境** — 測試固定生育率，或透過線性插值設定「最終目標生育率」
- **互動式儀表板：**
  - 含彩色閾值區間的撫養比軌跡折線圖
  - 即時人口金字塔（觀看它如何倒置，或對尼日爾／馬利而言如何持續擴底）
  - 各人口組成隨時間的絕對數量變化圖（Y 軸自動縮放）
- **引用資料來源** — 頁腳可展開，列出每個國家的資料來源
- **多語系（i18n）** — 完整翻譯：English、繁體中文、한국어、日本語
- **淺色 / 深色主題** 切換

## 快速開始

需要 **Node.js 18+**。

```bash
# 複製專案
git clone https://github.com/LawrenceHwang/taiwan-demographics.git
cd taiwan-demographics

# 安裝套件
npm install

# 啟動開發伺服器
npm run dev
```

在瀏覽器開啟 `http://localhost:5173`。

## 部署

### 方法 A — 手動（一行指令）

```bash
npm run deploy
```

此指令會執行 `vite build`，並將 `dist/` 資料夾推送至 `gh-pages` 分支，完成後發佈於：

```text
https://<your-username>.github.io/taiwan-demographics/
```

### 方法 B — GitHub Actions（推送後自動部署）

`.github/workflows/deploy.yml` 已包含在專案中。推送至 `main` 分支後即自動建置並部署。請在倉庫設定中啟用 GitHub Pages：

> **Settings → Pages → Source: GitHub Actions**

## 資料來源與研究方法

所有人口數據均為 2025 年估計值（適用時採用中位情境）。

| 資料來源 | 涵蓋範圍 |
| --- | --- |
| **[聯合國 WPP 2024](https://population.un.org/wpp/)** | 所有國家（主要參考） |
| **[台灣衛福部](https://www.mohw.gov.tw/)** + 國發會人口推估 2022–2070 | 台灣 |
| **[美國人口普查局 IDB 2024](https://www.census.gov/programs-surveys/international-programs/about/idb.html)** + CDC NCHS（TFR 2023：1.616） | 美國 |
| **[加拿大統計局](https://www.statcan.gc.ca/)** Cat. 91-520-X | 加拿大 |
| **[日本統計局](https://www.stat.go.jp/)** 2024 + 厚生勞動省人口動態統計（TFR 2023：1.20） | 日本 |
| **[韓國統計廳 KOSIS](https://kosis.kr/)** 2024（TFR 2023：0.72，歷史新低） | 韓國 |
| **[中國國家統計局](https://www.stats.gov.cn/)** 統計年鑑 2024（TFR 2023：1.09） | 中國 |
| **[德國聯邦統計局 Destatis](https://www.destatis.de/)** 人口預測 2024（TFR 2023：1.35） | 德國 |
| **聯合國 WPP 2024** + 世界銀行 WDI 2024 | 尼日爾、馬利 |

### 模擬方法論

- **死亡率：** 兩種曲線——*已開發國家*（預期壽命約 80–84 歲）與*高生育率國家*（預期壽命約 62 歲，嬰兒死亡率約 50‰）。依 WHO/UN 年齡別死亡率近似建模。
- **出生數：** 15–49 歲女性（約占該年齡層 50%）× (TFR ÷ 35 個生育年) = 年度出生數。
- **移民分配：** 均勻分配至 20–34 歲勞動年齡組；負值代表淨移出。
- **概念框架：** 撫養比閾值參考自 Max Fisher 的影片：*[How China blew up its own future](https://www.youtube.com/watch?v=AultJcNb90c)*。
